## 📌 Project Overview
This interactive Power BI dashboard analyzes Swiggy sales data to provide business insights through dynamic visualizations.

## 📊 Dashboard Features
- 📈 Total Revenue
- 🛒 Total Orders
- ⭐ Average Rating
- 🗺️ Revenue by State (India Shape Map)
- 📅 Monthly Revenue Trends
- 🍕 Top 5 Restaurants by Revenue
- 🏙️ Top 5 Cities by Orders
- 🍽️ Top 5 Dishes by Orders
- 📊 Rating Distribution
- 🔍 Interactive Filters & Reset Button

- Recommended category received the highest number of orders.
- January and August recorded the highest revenue.
